## üó∫Ô∏è Recommended Map to Download

**Download the official OSRS World Map and save it here as `OSRS-World-Map.png`**

### How to Download:

**Option 1: Official OSRS Website**
1. Visit: https://www.runescape.com/oldschool/world-map
2. Right-click the map image
3. Select "Save image as..."
4. Save to this folder as `OSRS-World-Map.png`

**Option 2: OSRS Wiki**
1. Visit: https://oldschool.runescape.wiki/w/World_map
2. Click "Full resolution image" link
3. Right-click and "Save image as..."
4. Save to this folder as `OSRS-World-Map.png`

**Option 3: Reddit Community**
1. Visit: https://www.reddit.com/r/OSRSmaps/
2. Find high-resolution map posts
3. Download and save here
